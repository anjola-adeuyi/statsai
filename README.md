# StatsAI

A landing page for StatsAI, a website analytics and sales platform.

**Live demo:** [https://statsai-six.vercel.app](https://statsai-six.vercel.app)

## What it does

StatsAI helps businesses understand their website performance and sales data. This repository contains the marketing site that explains the product, its features, and pricing.

## Tech stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **ShadCN UI** - Component library
- **next-themes** - Dark mode support
- **Unicorn Studio** - Interactive backgrounds

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project structure

- `app/` - Next.js app router pages and layouts
- `components/` - React components
  - `sections/` - Landing page sections (hero, features, pricing, etc.)
  - `ui/` - Reusable UI components
- `lib/` - Utility functions

## Features

- Responsive design
- Dark mode with theme toggle
- Interactive backgrounds via Unicorn Studio
- Animated testimonial marquees
- Pricing and FAQ sections

## Build

```bash
npm run build
npm start
```
